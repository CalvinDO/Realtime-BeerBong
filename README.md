# Realtime BeerPhong - Rendering Tests
Das ist ein Unterbranch zum Testen von verschiedenen Arten des Renderings in dem Projekt Realtime BeerPhong.
Hier wurde getestet, wie man Szenen auf Materialien rendert (RenderToMat -  Ordner). In dem Ordner "RenderToAR" testeten wir was für Auswirkungen das Post Processing auf den AR Modus hatte. In diesem Ordner ist zum Stand der Abgabe jedoch kein funktionsfähiges Beispiel vorhanden. 

Hier dem "Rootfolder" des Branches befindet sich das Realtime BeerPhong Projekt auf einem frühen Stand wo es über ThreeJS lief und ohne AR Möglichkeit. Das Abspielen der Anwendung in diesem Branch funktioniert wie in dem Main-Branch. Zum Testen der Anwendung empfehlen wir das Betriebssystem Windows und den Internetbrowser Chrome.

Um den Einfluss des Shaders zu testen kann man in der Datei ShaderVariables.js das "drunkLevel" zwischen 0.1 und 1.0 variieren sowie den Boolschen-Wert "drunk" auf true oder false setzen.
